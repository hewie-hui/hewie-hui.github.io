SSH方式配置GitHub

发表于 2019-08-04 | 分类于 教程 |
字数统计: 179
一、本地操作

打开git bash（我认为是在任意位置打开都可以，不过有的博主说是需要在你准备提交项目的文件夹下打开，可参考此博文链接）,输入以下命令：

git config --global user.name "yourname"

git config --global user.email "your@email.com"

ssh-keygen -t rsa -C "your@email.com"

出现以下信息按回车

一直按回车直到结束

系统会自动在.ssh文件夹下生成两个文件，id_rsa和id_rsa.pub，打开id_rsa.pub复制全部内容

二、GitHub操作

登陆账号setting–>SSH and GPG keys–> New SSH key

点击Add SSH key，输入命令ssh -T git@github.com出现以下内容就成功了
