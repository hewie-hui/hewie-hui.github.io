用hexo，github搭建轻量型博客

发表于 2019-08-03 | 分类于 教程 |
字数统计: 363
一、准备工作
1、注册一个GitHub账号,并创建一个名为你的用户名.github.io（比如hewie-hui.github.io）的仓库。
2、安装npm,node.js,git客户端,hexo

npm点击进入官网 安装命令：npm install npm@latest -g

nodejs点我下载

git客户端点我下载

hexo点击进入官网 安装命令：npm install hexo-cli -g

3、配置GitHub客户端

SSH方式配置本地与GitHub连接

二、初始化hexo
1、选一个位置新建一个文件夹(如：G:\hexo)用于hexo初始化

在该文件夹下执行命令： hexo init

hexo会自动下载一些文件到这个目录（截图是我后来截的，public文件夹此时应该还没有）

继续执行命令:

hexo g # 生成

hexo s # 启动服务预览

hexo就会在public文件夹生成相关html文件，这些文件将提交到github

此时访问http://localhost:4000可以看到一个默认主题的博客了

三、上传到GitHub
1、配置_config.yml文件

编辑_config.yml：

2、安装插件 npm install hexo-deployer-git --save
3、执行命令hexo d将文件提交到GitHub

此时访问你的用户名.github.io（比如hewie-hui.github.io）就能看到博客了

四、写博客

在hexo根目录执行命令 hexo new ‘test-blog’`

-posts文件夹下就生成我们的博客文件了

编辑.md文件即编辑我们的博客
